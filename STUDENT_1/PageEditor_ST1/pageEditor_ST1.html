The splash scrren was developed using HTML and Css wwith an emphasis on animation, multimedia itegration, and responsiveness. Video is muted and looped to avoid audio distractions, and the overlay ensures sufficient contrast between text and background video. Volunteer Page

Technical Description

The Volunteer page uses a clean, responsive layout structured with semantic HTML elements like <section>, <article>, and <main>. It features a fixed header with a logo and navigation bar styled using Flexbox. Each volunteer opportunity is contained in a "project card" within a responsive CSS grid, adapting to different screen sizes via grid-template-columns and auto-fit.

The CSS design implements modern aesthetics using card shadows, hover effects, rounded corners, and vibrant color highlights. Animations such as scale-on-hover are applied using transform and transition properties. User engagement is enhanced with a testimonials section styled via box-shadow and feedback forms that use emoji-based ratings.

Icons from Font Awesome and emoji symbols add visual clarity and appeal. A fixed-width .wrapper centers the content while preventing overflow issues, and visual balance is ensured through consistent spacing, margins, and color theming across sections.

Accessibility

Accessibility was incorporated by ensuring all images (like the logo) include alt attributes. Font sizes are readable with good contrast against the background, especially in headers and cards. The layout uses semantic tags (e.g., <section>, <h2>, <p>) to support screen readers. Interactive elements like feedback buttons and rating inputs are styled with sufficient padding and focus states for better usability.

Icons are paired with descriptive labels for clarity, and the structure allows for keyboard navigation. Additionally, color cues are supported by visual shapes (buttons, borders, emojis) to avoid color-only communication.

Challenges and Lessons Learned

During the development of the webpages, several challenges were encountered that contributed to valuable learning experiences:

  • Maintaing Layout Consistency:One of the main challenges was ensuring consistent layout and styling across all pages. This required careful use of CSS file and standard class naming conventions. I learned the importance of a modular CSS struncture and reusing components wherever possible.

  • Responsive Design:Making the site mobile-friendly across different screen sizes was difficult initially. With practice using media queries and CSS Grid/Flexbox, I learned how to create layouts that adjust fluidly to various devices.

  • Embedding Multimedia:Integrating a full-screen video background in the splash page presented preformance and layering issues. I learned how to user z-index properly effectively along with object-fit and position techniques to maintaines visuaal hierarchy.

  • Validation Errors:I encountered several validation errors to improper nesting of elements. Fixing these errors helped me better understand HTML structure rules.

Overall, this project improved my understanding of front-end web development fundementals, including HTML5 semantics, CSS styling, responsiveness, and accessibility.
